### **Nếu không load được:**
1. **Check browser**: Dùng Chrome/Firefox/Safari latest
2. **Clear cache**: Cmd+Shift+R (Mac) hoặc Ctrl+F5 (PC)
3. **Check console**: F12 → Console tab → check errors
